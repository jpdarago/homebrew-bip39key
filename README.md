# Jpdarago Bip39key

## How do I install these formulae?

`brew install jpdarago/bip39key/<formula>`

Or `brew tap jpdarago/bip39key` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "jpdarago/bip39key"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
